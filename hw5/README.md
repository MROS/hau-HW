	基本思路：將一個div製成方形，以許多方形來連成貪食蛇的身體、以及貪食蛇的食物

	- 能夠用鍵盤控制上下左右
	- 能夠吃食物變長
	- 撞到自己身體或障礙物就遊戲結束
	- 支援穿牆
